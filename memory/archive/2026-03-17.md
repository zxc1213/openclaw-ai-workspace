## 经验教训
- systemd service 文件被 gateway 自动管理，改 Environment 要用 drop-in 目录
- SIGUSR1 是热重启，不重新加载 systemd 配置
- JS let/const 有 TDZ，IIFE 和变量声明顺序要注意
- OpenClaw subagent session 默认 60 分钟自动归档，归档后不出现在 CLI 列表中
