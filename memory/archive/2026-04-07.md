## OpenCLI 自定义适配器：智谱费用账单

- **时间**: 23:00 ~ 23:40
- **背景**: 通过 OpenCLI 学习，尝试为 bigmodel.cn 创建自定义适配器
- **最终产出**: 4 个 YAML 适配器文件
  - `bill.yaml` — 费用明细（按日）
  - `summary.yaml` — 按模型汇总
  - `bykey.yaml` — 按 API Key 分组
  - `detail.yaml` — 按日期→API Key→模型三级分组（最终版）
- **安装位置**: `~/.opencli/clis/bigmodel/`
- **命令**: `opencli bigmodel detail --month 2026-04`

### 踩坑记录
1. **API 方法错误**: 文档说是 POST，实际是 GET
2. **参数名不对**: 参数名是 `billingMonth` 不是 `month`
3. **认证方式**: 需要 Cookie 中的 `bigmodel_token_production` JWT 做 Authorization header
4. **响应字段**: 返回在 `j.rows`（不是 `data.data`），字段名用 camelCase
5. **分页**: pageSize=100，循环拉取直到返回空或少于 pageSize

### 技术要点
- OpenCLI 适配器通过 `navigate` 到目标网站获取浏览器 Cookie，后续 `fetch` 用 `credentials: 'include'`
- `evaluate` 块中可以写任意 async JavaScript，支持分页循环
- `map` 块做字段映射和列选择
- Cookie 认证场景比 API Key 认证复杂，需要从 `document.cookie` 手动提取 JWT
