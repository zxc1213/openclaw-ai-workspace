### QQ Bot 主动消息通知
- 新增 `sendQQNotification()` — 通过 QQ 开放平台 API 发 C2C 主动消息
- Token 获取：`POST https://bots.qq.com/app/getAppAccessToken`
- 发消息：`POST https://api.sgroup.qq.com/v2/users/{openid}/messages`
- 消息体：`{ content: text, msg_type: 0 }`（0=纯文本）
- Token 自动缓存 + 提前刷新
- 集成到 `addAlert()` 通知链（与 Server酱、Telegram、Webhook 并列）
