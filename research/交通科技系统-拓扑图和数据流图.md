# 拓扑图 + 数据流图（Mermaid 源码）

## 方式一：应用网络拓扑图

```mermaid
graph TB
    subgraph Internet["🌐 互联网（DMZ区）"]
        A["广东省交通科技协同创新信息平台<br/>省级门户"]
    end

    subgraph VPN_Zone["🔐 安全网关 / API网关"]
        GW["防火墙 + WAF + 安全数据交换"]
    end

    subgraph Intranet["🏢 集团专网"]
        B["集团科技协同管理平台<br/>集团管理中枢"]
    end

    subgraph Sub1["华路科技内部网络"]
        C["华路科技管理平台<br/>子公司业务系统"]
    end

    subgraph Sub2["建设公司内部网络"]
        D["建设公司科技管理平台<br/>子公司业务系统"]
    end

    User1["👤 公众用户<br/>科研机构/高校/社会公众"]
    User2["👔 集团内部用户<br/>集团总部/合作单位"]
    User3["🔧 子公司用户<br/>华路科技/建设公司"]

    User1 -->|HTTPS| A
    A <-->|SSL/TLS 加密| GW
    GW <-->|安全数据交换| B
    User2 -->|专网直连| B
    B <-->|专线/VPN| C
    B <-->|专线/VPN| D
    User3 -->|内部网络| C
    User3 -->|内部网络| D
    User3 -.->|互联网| A

    style Internet fill:#E3F2FD,stroke:#1565C0,stroke-width:2px
    style Intranet fill:#E8F5E9,stroke:#2E7D32,stroke-width:2px
    style Sub1 fill:#FFF3E0,stroke:#E65100,stroke-width:1px
    style Sub2 fill:#FFF3E0,stroke:#E65100,stroke-width:1px
    style VPN_Zone fill:#FFEBEE,stroke:#C62828,stroke-width:2px
```

## 方式二：数据流向图

```mermaid
graph LR
    subgraph 省级平台["广东省交通科技协同创新信息平台"]
        P["省级平台"]
    end

    subgraph 集团平台["集团科技协同管理平台"]
        G["集团平台"]
    end

    subgraph 华路["华路科技管理平台"]
        H["华路平台"]
    end

    subgraph 建设["建设公司科技管理平台"]
        J["建设平台"]
    end

    %% 下行数据流
    P ==>|行业政策<br/>科技资讯<br/>科研任务| G
    G ==>|标准化推送| H
    G ==>|标准化推送| J

    %% 上行数据流
    H -.->|科研数据上报| G
    J -.->|科研数据上报| G
    G -.->|汇总数据上报| P

    %% 跨系统协同
    P <..>|项目申报<br/>成果评审| H
    P <..>|项目申报<br/>成果评审| J

    %% 集团成果上报
    G -->|集团成果<br/>面向公众展示| P

    style 省级平台 fill:#E3F2FD,stroke:#1565C0,stroke-width:2px
    style 集团平台 fill:#E8F5E9,stroke:#2E7D32,stroke-width:2px
    style 华路 fill:#FFF3E0,stroke:#E65100,stroke-width:1px
    style 建设 fill:#FFF3E0,stroke:#E65100,stroke-width:1px
```

## 方式三：用户访问关系图

```mermaid
graph TB
    U1["👤 公众用户"]
    U2["👔 集团内部用户"]
    U3["🔧 子公司用户"]

    U1 -->|"互联网直接访问<br/>HTTPS"| A["广东省交通科技协同创新<br/>信息平台"]
    U2 -->|"集团专网直连"| B["集团科技协同<br/>管理平台"]
    B <-->|"平台对接"| A
    U3 -->|"路径一：内部系统"| C["华路科技/<br/>建设公司管理平台"]
    C -->|"数据对接"| B
    U3 -.->|"路径二：互联网"| A

    style A fill:#E3F2FD,stroke:#1565C0,stroke-width:2px
    style B fill:#E8F5E9,stroke:#2E7D32,stroke-width:2px
    style C fill:#FFF3E0,stroke:#E65100,stroke-width:1px
```

---

### 使用方法

1. **Draw.io（推荐）**：打开 https://app.diagrams.net → Arrange → Insert → Advanced → Mermaid，粘贴代码
2. **Mermaid Live Editor**：打开 https://mermaid.live 粘贴代码，可导出 PNG/SVG
3. **WPS**：新版 WPS 支持插入 Mermaid 图表（插入 → 智能图表），或截图粘贴
4. **导出为图片**后可直接插入 Word 文档的"拓扑图"章节
