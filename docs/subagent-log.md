# Subagent 任务日志

> 每次派发 subagent 任务时追加一条记录，定期统计成功率。

## 日志格式

```markdown
| 日期 | 任务类型 | agent | 结果 | 耗时 | 备注 |
```

## 任务类型分类

- **coding**: 代码编写/修改/重构
- **review**: 代码审查/安全审计
- **research**: 技术调研/资料收集
- **doc**: 文档撰写/整理
- **admin**: 系统管理/配置变更
- **fix**: Bug 修复/排障

## 结果分类

- ✅ **success**: 一次完成，无需返工
- ⚠️ **partial**: 部分完成，需补充
- ❌ **failed**: 失败/需重做
- 🔄 **retried**: 重试后成功
