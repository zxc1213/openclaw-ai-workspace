## 2 软件概述

### 2.1 功能简介

河南交投集团科技创新服务平台是面向河南交通投资集团有限公司及下属各单位的一站式科技管理信息化平台。平台采用微服务架构，集成了科技项目全生命周期管理、科技成果管理、专家资源管理、简报统计分析、会议管理、社区交流、创新成果推广及科技超市等核心功能，实现了科技工作的数字化、标准化和规范化管理。

平台主要功能包括：

- **项目申报与管理：** 覆盖科技项目从申报、审批、立项、合同签订、开题、执行、中期检查、验收到结题的全生命周期管理
- **科技成果管理：** 支持论文、专利、软件著作权、标准、工法、科技奖励、技术规程、研究报告等8类科技成果的登记、审核与管理
- **专家管理：** 建立集团级专家资源库，支持专家信息维护、分类管理和专家抽取
- **简报统计：** 提供多维度的数据统计与分析功能，支持科技简报自动生成
- **会议管理：** 支持科技类会议的计划、通知、签到及纪要管理
- **社区交流：** 为科研人员提供在线交流、问题讨论和经验分享的协作空间
- **创新成果推广：** 搭建科技成果展示与推广平台，促进成果转化应用
- **科技超市：** 提供技术需求发布与对接服务，促进技术合作

### 2.2 适用范围

本平台适用于河南交投集团总部及下属各单位的科技管理工作，主要服务对象包括：

- 集团总部科技管理部门
- 各下属单位科技管理部门
- 科研项目负责人及团队成员
- 集团内外聘专家
- 各类科研参与人员

## 3 安装与部署

### 3.1 系统部署说明

本平台采用微服务架构部署，由专业技术人员负责服务器端部署和运维。部署过程包括以下主要步骤：

1. **环境准备：** 在服务器上安装JDK 1.8、MySQL 8.0、Redis、RabbitMQ等基础软件
2. **微服务部署：** 部署注册中心（Eureka）、网关服务（Gateway）及各业务微服务
3. **数据库初始化：** 执行数据库初始化脚本，创建数据库表结构和初始数据
4. **配置调整：** 根据实际环境修改配置文件中的数据库连接、缓存连接等参数
5. **服务验证：** 启动所有微服务，验证各模块功能正常运行

### 3.2 客户端访问

平台采用B/S（浏览器/服务器）架构，用户无需安装客户端软件，通过浏览器访问平台地址即可使用。打开浏览器，输入平台网址，即可进入平台登录页面。

**推荐浏览器：**
- Google Chrome 80及以上版本
- Mozilla Firefox 75及以上版本
- Microsoft Edge 80及以上版本

## 4 功能操作说明

### 4.1 用户登录

平台提供三种登录方式，用户可根据自身情况选择合适的方式登录系统。

#### 4.1.1 账号密码登录

1. 打开浏览器，访问平台地址，进入登录页面
2. 在登录页面选择"账号密码登录"标签
3. 在"账号"输入框中输入系统分配的用户名或手机号码
4. 在"密码"输入框中输入登录密码
5. 输入页面显示的验证码（如有）
6. 点击"登录"按钮
7. 登录成功后，系统自动跳转至工作台首页

**首次登录提示：** 首次登录系统后，系统会提示用户修改初始密码。请按照密码强度要求（不少于8位，需包含字母和数字）设置新密码。

#### 4.1.2 短信验证码登录

1. 打开浏览器，访问平台地址，进入登录页面
2. 在登录页面选择"短信登录"标签
3. 在"手机号"输入框中输入已注册的手机号码
4. 点击"获取验证码"按钮
5. 在手机上查收短信验证码
6. 在"验证码"输入框中输入收到的验证码
7. 点击"登录"按钮

**注意事项：**
- 每日短信验证码发送次数有限制，请勿频繁获取
- 验证码有效时间为5分钟，超时需重新获取
- 如未收到验证码，请检查手机信号或联系系统管理员

#### 4.1.3 微信扫码登录

1. 打开浏览器，访问平台地址，进入登录页面
2. 在登录页面选择"微信登录"标签
3. 页面显示微信登录二维码
4. 使用微信手机客户端扫描二维码
5. 在手机端确认授权登录
6. 系统验证通过后自动跳转至工作台首页

**注意事项：**
- 微信登录需提前在系统中绑定微信号
- 首次使用微信登录时，系统会引导用户完成账号绑定

### 4.2 工作台/首页

#### 4.2.1 首页概览

登录成功后进入工作台首页，首页采用模块化布局，为用户提供一站式信息概览：

- **待办事项：** 显示当前用户需要处理的待办任务，如待审批项目、待审核成果等，点击可直接跳转至对应处理页面
- **通知公告：** 展示平台发布的最新通知和公告信息，支持按时间排序查看
- **数据统计卡片：** 以图形化方式展示关键统计数据，包括项目总数、在研项目数、成果数量、专家人数等
- **快捷入口：** 提供常用功能的快速访问入口，方便用户一键进入常用模块

#### 4.2.2 个人中心

在页面右上角点击用户头像，可进入个人中心，功能包括：

- **个人信息维护：** 修改个人基本信息，如姓名、联系方式、所属单位等
- **密码修改：** 修改登录密码
- **消息通知设置：** 配置消息通知方式和通知范围
- **微信绑定/解绑：** 管理微信账号与系统账号的绑定关系
- **操作日志：** 查看个人操作记录

### 4.3 项目申报与管理

项目申报与管理是平台的核心功能模块，实现了科技项目从申报到结题的全流程闭环管理。

#### 4.3.1 项目申报

项目负责人通过以下步骤进行项目申报：

1. 在左侧导航栏中点击"项目管理"→"项目申报"
2. 点击"新建申报"按钮，进入项目申报表单页面
3. 按照表单要求依次填写以下信息：
   - **基本信息：** 项目名称、项目类别（科研项目/技改项目/新技术应用等）、申报单位、项目负责人、项目周期
   - **项目简介：** 项目背景、研究目的、主要研究内容（不超过2000字）
   - **预期成果：** 填写项目预期产生的各类成果，如论文、专利、标准等
   - **经费预算：** 填写项目总预算及各年度预算明细
   - **项目组成员：** 添加项目团队成员信息，包括姓名、职务、分工
4. 上传附件材料：项目申报书、可行性研究报告等必要文档
5. 确认信息无误后，点击"提交申报"按钮

**注意事项：**
- 申报表单中标"*"号的为必填项，必须填写完整方可提交
- 附件上传支持doc、docx、pdf格式，单个文件不超过50MB
- 提交后如需修改，需联系审批人员退回后重新编辑

#### 4.3.2 项目审批

审批人员（集团科技管理员、单位科技管理员等）进行项目审批：

1. 在工作台首页的"待办事项"中，或通过"项目管理"→"项目审批"进入审批列表
2. 点击待审批项目，查看项目申报详细信息
3. 逐项审核申报内容，查看上传的附件材料
4. 在审批意见栏中填写审批意见
5. 选择审批结果："通过"、"退回修改"或"驳回"
6. 点击"提交审批"完成操作

**审批流程说明：**
- 项目审批实行多级审批制，依次经过单位科技管理员审核、单位总工审核、集团科技管理员审核、集团分管领导审批
- 每一级审批通过后，项目自动流转至下一级审批人
- 退回修改时，项目退回至申报人，申报人修改后可重新提交
- 驳回时项目终止流程，需重新申报

#### 4.3.3 项目立项

审批通过后进入立项阶段：

1. 集团科技管理员在"项目管理"→"项目立项"中，选择已审批通过的项目
2. 分配项目编号，确认项目经费
3. 下发项目立项通知书
4. 项目状态变更为"已立项"

#### 4.3.4 合同签订

1. 项目立项后，项目负责人需在系统中填写合同信息
2. 上传合同扫描件
3. 单位科技管理员审核合同内容
4. 审核通过后，合同状态更新为"已签订"

#### 4.3.5 开题报告

1. 项目负责人在"项目管理"→"开题管理"中提交开题报告
2. 填写开题报告表单，包括研究方案、技术路线、进度安排等
3. 上传开题报告文档
4. 审批人员审核开题报告

#### 4.3.6 项目执行

项目进入执行阶段后，项目负责人需定期维护项目进展：

1. **进度填报：** 按季度或月度填写项目进展报告，记录研究进展、阶段性成果、存在的问题等
2. **经费使用：** 记录项目经费使用情况，上传报销凭证
3. **成果登记：** 在项目执行过程中产生的科技成果，可实时在系统中登记
4. **人员变更：** 如项目组成员发生变动，需在系统中提交变更申请

#### 4.3.7 中期检查

1. 系统根据项目计划自动提示中期检查时间
2. 项目负责人提交中期检查报告
3. 审批人员组织中期检查评审
4. 根据评审结果，项目可继续执行、整改或终止

#### 4.3.8 项目验收

1. 项目负责人在项目完成后提交验收申请
2. 填写验收材料，包括项目总结报告、成果清单、经费决算等
3. 上传相关附件材料
4. 管理人员组织专家评审
5. 评审通过后，项目状态更新为"已验收"

#### 4.3.9 项目结题

1. 验收通过后，项目负责人在系统中提交结题申请
2. 补充结题相关材料
3. 管理人员审核结题材料
4. 审核通过后，项目状态更新为"已结题"，项目全流程完成

#### 4.3.10 项目查询

系统提供灵活的项目查询功能：

1. 进入"项目管理"→"项目查询"
2. 在查询条件区域可按以下条件筛选：
   - 项目名称（支持模糊搜索）
   - 项目状态（申报中/已立项/执行中/已结题等）
   - 项目类别
   - 申报单位
   - 项目负责人
   - 时间范围
3. 点击"查询"按钮显示符合条件的项目列表
4. 点击项目名称可查看项目详细信息

### 4.4 科技成果管理

科技成果管理模块支持8类科技成果的登记、审核与管理。

#### 4.4.1 支持的成果类型

平台支持以下8类科技成果的管理：

1. **论文：** 学术论文、会议论文等
2. **专利：** 发明专利、实用新型专利、外观设计专利
3. **软件著作权：** 软件著作权登记
4. **标准：** 国家标准、行业标准、地方标准、企业标准
5. **工法：** 施工工法
6. **科技奖励：** 各类科技奖项
7. **技术规程：** 技术规程、操作指南等
8. **研究报告：** 科研报告、调研报告等

#### 4.4.2 成果登记

用户通过以下步骤登记科技成果：

1. 在左侧导航栏点击"成果管理"→"成果登记"
2. 点击"新增成果"按钮
3. 选择成果类型
4. 根据不同成果类型，填写对应的登记表单。各类型通用信息包括：
   - 成果名称
   - 所属项目（可选择关联的科研项目）
   - 第一完成人
   - 完成单位
   - 完成日期
   - 成果简介
   - 成果附件（论文PDF、专利证书扫描件等）
5. 各类型特有信息举例：
   - **论文：** 期刊名称、ISSN号、卷期号、影响因子、收录情况
   - **专利：** 专利号、专利类型、授权日期、权利人
   - **标准：** 标准编号、标准级别、发布日期、实施日期
6. 填写完毕后点击"提交"按钮

#### 4.4.3 成果审核

审核人员对提交的成果进行审核：

1. 进入"成果管理"→"成果审核"
2. 查看待审核成果列表
3. 点击成果名称查看详细信息和附件材料
4. 核实成果信息的真实性和完整性
5. 填写审核意见，选择"通过"或"退回"
6. 审核通过后，成果纳入平台成果库

#### 4.4.4 成果查询与统计

1. 进入"成果管理"→"成果查询"
2. 可按成果类型、完成单位、时间范围、所属项目等条件检索
3. 系统提供成果统计图表，展示各类成果的数量、趋势分布
4. 支持导出成果统计报表

### 4.5 专家管理

#### 4.5.1 专家信息维护

1. 系统管理员进入"专家管理"→"专家库"
2. 点击"新增专家"按钮
3. 填写专家基本信息：
   - 姓名、性别、出生年月
   - 工作单位、职务、职称
   - 学历、学位、毕业院校及专业
   - 研究方向、专业特长
   - 联系电话、电子邮箱
   - 主要业绩和代表性成果
4. 选择专家分类标签（技术领域、专业方向）
5. 保存专家信息

#### 4.5.2 专家分类管理

1. 在"专家管理"→"专家分类"中可维护专家分类体系
2. 支持多级分类，如"道路工程→路基路面""桥梁工程→桥梁检测"等
3. 可为每个分类设置专家评审权重

#### 4.5.3 专家抽取

在进行项目评审时，管理人员可进行专家抽取：

1. 进入"专家管理"→"专家抽取"
2. 设置抽取条件：专业领域、评审类型、回避单位等
3. 系统根据条件从专家库中随机抽取候选专家
4. 管理人员确认抽取结果并发送评审邀请

### 4.6 简报统计

#### 4.6.1 数据统计

1. 进入"简报统计"→"数据统计"页面
2. 系统提供以下统计维度：
   - 项目统计：按年度、类别、单位统计项目数量和金额
   - 成果统计：按类型、年度、单位统计各类成果数量
   - 经费统计：项目经费投入、使用情况统计
   - 专家统计：专家数量、领域分布统计
3. 统计结果以表格和图形（柱状图、饼图、折线图）方式展示
4. 支持按时间范围筛选数据

#### 4.6.2 简报生成

1. 进入"简报统计"→"简报管理"
2. 点击"生成简报"按钮
3. 选择简报周期（月报/季报/年报）和时间范围
4. 选择需要纳入简报的统计模块
5. 系统自动生成简报，包含数据表格、图表和分析文字
6. 管理人员可编辑完善简报内容

## 5 常见问题与注意事项

### 5.1 常见问题

**Q1：忘记登录密码怎么办？**

A：在登录页面点击"忘记密码"链接，通过手机短信验证身份后可重置密码。也可联系本单位系统管理员进行密码重置。

**Q2：提交的申报被退回，如何修改？**

A：在"项目管理"→"我的申报"中找到被退回的项目，点击"编辑"按钮查看退回意见，根据意见修改后重新提交。

**Q3：上传附件失败怎么办？**

A：请检查文件格式是否符合要求（支持doc/docx/pdf格式），文件大小是否超过限制（单个文件不超过50MB）。如仍有问题，请尝试更换浏览器或清除浏览器缓存后重试。

**Q4：如何查看项目审批进度？**

A：在"项目管理"→"我的申报"中，点击项目名称进入详情页，可查看当前审批节点和审批进度。

**Q5：微信登录无法使用怎么办？**

A：请确认已绑定微信号。如未绑定，请先用账号密码登录后在个人中心完成微信绑定。如绑定后仍无法使用，请联系系统管理员。

### 5.2 注意事项

1. **账号安全：** 请妥善保管登录账号和密码，不要将账号借予他人使用。建议定期修改密码，密码长度不少于8位，包含字母和数字的组合。

2. **数据准确性：** 在系统中填报的信息应确保真实准确，尤其涉及项目经费、成果信息等重要数据。

3. **及时处理待办：** 请及时查看并处理工作台中的待办事项，避免影响项目进度和审批流程。

4. **附件规范：** 上传的附件文档应命名清晰，便于查阅。重要文档建议上传PDF格式。

5. **浏览器兼容性：** 建议使用Google Chrome浏览器以获得最佳使用体验，不建议使用IE浏览器。

6. **数据备份：** 重要数据请自行做好备份，系统虽定期备份，但建议用户定期导出关键数据。

7. **版权声明：** 在社区交流和成果推广中发布的内容，应确保不侵犯他人知识产权。
